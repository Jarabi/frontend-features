# Frontend Features

A production-grade landing page showcasing 6 frontend features built with React and Vite.

## Features

- **Hero Section** — Animated badge, stats display, and call-to-action buttons
- **Search** — Real-time feature search with debounced input
- **Tag Filtering** — Filter features by category (Performance, UX, Accessibility, etc.)
- **Feature Cards** — Hover effects, difficulty badges, and navigation to each feature
- **Responsive Design** — Mobile-friendly layout with adaptive grid

## Available Features

| # | Feature | Description | Difficulty |
|---|---------|-------------|------------|
| 1 | Infinite Scrolling | IntersectionObserver, lazy loading, pagination | Intermediate |
| 2 | Debounced Search | 300ms debounce, caching, text highlighting | Intermediate |
| 3 | Loading Skeletons | Shimmer animations, multiple layouts | Beginner |
| 4 | Toast Notifications | Multiple types, stacking, auto-close | Beginner |
| 5 | Modal Dialogs | Focus trap, ESC close, scroll lock | Advanced |
| 6 | Drag and Drop | Touch support, keyboard accessibility | Advanced |

## Usage

```bash
cd 0-landing-page
npm install
npm run dev
```

## Tech Stack

- React 19
- Vite
- CSS (no external UI libraries)
